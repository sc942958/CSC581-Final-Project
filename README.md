# CSC581-Final-Project: Online Connect 4
<img width="640" height="500" alt="Blank diagram(2)" src="https://github.com/user-attachments/assets/018cddc6-1ffe-464d-9f69-6d51c33dbf51" />

My Project is a multiplayer online Connect 4 server. The application will allow 2 players to play a game of Connect 4 online, with the server being able to host more than 1 match at a time. The backend server will use the Nodejs base image, and the frontend will use Expressjs.
